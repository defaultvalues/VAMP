# Vector Approximate Message Passing

A VAMP implementation with detailed algorithm dervation. The procedure follows: https://arxiv.org/abs/1610.03082. 

You can run the demo in `VAMP.py`, and the derivation written by myself is `vamp.pdf`. I'm happy to know if you use this code in your research, and please let me know if you have any questions about the code or the derivation.
